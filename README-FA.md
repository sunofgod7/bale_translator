# ربات مترجم بله

ربات مترجم انگلیسی به فارسی برای پیام‌رسان بله

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها
```bash
npm install
```

### 2. تنظیم متغیرهای محیطی
فایل `.env.local` را ویرایش کنید:
```
BALE_TOKEN=توکن_ربات_شما
GEMINI_API_KEY=کلید_API_جمینی_شما
PORT=8080
```

### 3. اجرای محلی
```bash
npm start
```

## دیپلوی روی Railway

### 1. متغیرهای محیطی را در Railway تنظیم کنید:
- `BALE_TOKEN`: توکن ربات بله
- `GEMINI_API_KEY`: کلید API گوگل جمینی
- `PORT`: 8080 (اختیاری)

### 2. منتظر بمانید تا دیپلوی کامل شود

### 3. وب‌هوک را تنظیم کنید:

#### روش اول: استفاده از اسکریپت Node.js
```bash
node set-webhook.js
```

#### روش دوم: استفاده از PowerShell
```powershell
.\set-webhook.ps1
```

#### روش سوم: استفاده از مرورگر
در مرورگر باز کنید:
```
https://tapi.bale.ai/bot[TOKEN]/setWebhook?url=https://[DOMAIN]/api/webhook
```

### 4. بررسی وضعیت وب‌هوک
```
https://tapi.bale.ai/bot[TOKEN]/getWebhookInfo
```

## بررسی سلامت سرویس

برای اطمینان از اینکه سرور در حال اجرا است:
```
https://baletranslator-production.up.railway.app/health
```

باید پاسخ `{"status":"ok"}` را دریافت کنید.

## رفع مشکلات

### خطای 503 Service Unavailable
- منتظر بمانید تا دیپلوی Railway کامل شود (2-3 دقیقه)
- لاگ‌های Railway را بررسی کنید
- مطمئن شوید که متغیرهای محیطی درست تنظیم شده‌اند

### خطای Syntax در دیپلوی
- مطمئن شوید که آخرین نسخه کد را push کرده‌اید
- بررسی کنید که `node -c api/webhook.js` بدون خطا اجرا شود

### ربات پاسخ نمی‌دهد
- وب‌هوک را بررسی کنید: `getWebhookInfo`
- لاگ‌های Railway را بررسی کنید
- مطمئن شوید که URL وب‌هوک صحیح است
